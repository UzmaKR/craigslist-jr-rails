Craigslist-jr-rails

Created a Craigslist-type web app while learning the Rails framework.

View project onGitHub

# Summary: This was my first attempt to create the front/back-end of an app in order to learn the Rails framework. The app works similar to a Craiglist website: with categories of listings, and with each listing created by a user. One key feature is that listings can only be edited by its author; this is accomplished by using a special key in the URL. Only the author knows the key and can go to the edit URL.

Gems used:

  • 'pg' for PostgreSQL
  • 'faker' for Faker gem to generate fake user names, posts, comments.

Authors: Uzma Rahman, Phillip Mispagel
